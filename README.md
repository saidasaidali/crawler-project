# Crawler Project

This is a web crawler project with a backend API and a frontend dashboard.

## Features

- Web crawling functionality
- Chatbot integration
- Dashboard for monitoring
- Database storage
- Scheduled tasks

## Installation

1. Clone the repository:
   ```
   git clone <repository-url>
   cd crawler_project
   ```

2. Install dependencies:
   ```
   pip install -r requirements.txt
   ```

3. Run the backend:
   ```
   python backend/app.py
   ```

4. Open the frontend in a browser (e.g., frontend/home.html)

## Usage

- Access the dashboard at `frontend/dashboard.html`
- Use the chatbot at `frontend/chatbot.html`
- Search functionality at `frontend/search.html`

## Project Structure

- `backend/`: Contains the Python backend code
- `frontend/`: Contains the HTML, CSS, and JS frontend files
- `requirements.txt`: Python dependencies
